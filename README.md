## ゲームシステム

- **インフレは止まらない**：放置するとふくらみ続け、限界サイズで1秒経つと過熱崩壊
- **リスクプレミアム**：棘スレスレをかすめるとボーナス＋コンボ
- **🌈 バブルフィーバー**：ゲージMAXで12秒間 資産3倍・配当2倍
- **📊 投資戦略ドラフト**：四半期（18秒）ごとに9種のパークから3択（保険🛡️・高配当🪙・省エネ噴射💨…）
- **💨 風アクション**：噴射の風でトゲを吹き飛ばすと、味方の誘導弾に変わる
- **🐻 ボス「ベア（空売りの主）」**：第4四半期ごとに襲来。逃げながら風でトゲをぶつけて撃退
- **💹 市場イベント**：好景気・金融緩和・空売り規制がランダム発生
- **👑 称号**：最終資産に応じて「🌱 養分」〜「🫧 バブルの神」
