# 🎈 BUBBLENOMICS

**〜バブルは、はじける直前がいちばん儲かる〜**

弾幕回避 × バブル経済チキンレース × ローグライトドラフト。
あなたは「バブル経済」という名の風船。ふくらむほど資産が **サイズ²** で爆増しますが、当たり判定も大きくなり、市場の棘に触れた瞬間バブル崩壊します。

## ゲームシステム

- **インフレは止まらない**：放置するとふくらみ続け、限界サイズで1秒経つと過熱崩壊
- **リスクプレミアム**：棘スレスレをかすめるとボーナス＋コンボ（微スローモーション付き）
- **🌈 バブルフィーバー**：ゲージMAXで12秒間 資産3倍・配当2倍・専用BGM
- **📊 投資戦略ドラフト**：四半期（18秒）ごとに12種のパークから3択（保険🛡️・複利💹・火事場の強欲🔥…）
- **💨 風アクション**：噴射の風でトゲを吹き飛ばすと、味方の誘導弾に変わる
- **🐻🦅 2種のボス**：「ベア（空売りの主）」と「タカ（利上げの主）」が第4四半期ごとに交互襲来。タカの利上げオーラ圏内では資産がしぼむ
- **🌀⚡ 進化するハザード**：サボテン爆弾・トゲの壁・吸引渦・当局の監視ビーム
- **💹 市場イベント**：好景気・金融緩和・空売り規制がランダム発生
- **🌦️ 相場の気分**：ランごとに6種のモディファイア（強気相場・乱気流・投機熱…）
- **📅 本日の相場**：日替わりシードで毎日同じ相場に挑戦、専用ベスト記録
- **🏆 実績16種**：永続保存。💎ダイヤ回収、タカ狩り、第8四半期の生還者…
- **👑 称号**：最終資産に応じて「🌱 養分」〜「🫧 バブルの神」。結果はワンタップでコピーしてシェア可能
- **🎵 プロシージャルBGM**：タイトル／プレイ中／ボス戦／フィーバーで曲調が変化（全てWebAudioでコード生成）
- **🎨 風船スキン5色**：実績数に応じて解放（ミント・ソーダ・ゴールド・オニキス）。タイトルで切り替え
- **📜 日替わり相場格言**：タイトルに毎日ひとこと（「靴磨きの少年が相場を語り出したら、売り。」…）

## 技術

- 外部アセット・依存ライブラリ・ビルドツール **ゼロ**。ブラウザネイティブで動作
- Canvas 2D + WebAudio API（効果音もすべてコード生成）
- ハイスコア・実績・設定は localStorage に保存

## ファイル構成

役割ごとにファイルを分割しています（`index.html` が下記の順で読み込み、同一グローバルスコープを共有）。

```
index.html          … HTMLシェル（CSS + 各スクリプトを読み込むだけ）
css/
  style.css         … スタイル
src/
  setup.js          … キャンバス初期化
  utils.js          … 数学ヘルパー・乱数・localStorageラッパ・フォント
  config.js         … バランス調整用の定数（膨張・風・ボス・オーラ 等）
  data.js           … データテーブル（パーク／市場イベント／実績／気分／称号／ニュース）
  state.js          … ゲーム状態・エンティティ配列・背景の塵
  audio.js          … WebAudio 効果音
  input.js          … ポインタ／キーボード入力・共有／結果コピー
  flow.js           … 開始・被弾・崩壊・フィーバー・ドラフト
  boss.js           … ボス（ベア／タカ）のAIと攻撃
  effects.js        … パーティクル放出・ポップアップ・ボタン描画の共通処理
  spawn.js          … 敵・アイテムの出現ロジック
  simulation.js     … メイン更新（simulate）とエンティティ更新
  render-world.js   … ワールド描画（背景・エンティティ・風船・ボス）
  render-ui.js      … UI描画（HUD・ドラフト・タイトル・リザルト・オーバーレイ）
  main.js           … ゲームループ・カーソル制御・デバッグフック
```

デプロイは GitHub Actions（`.github/workflows/deploy.yml`）でリポジトリをそのまま GitHub Pages へ公開しています。
